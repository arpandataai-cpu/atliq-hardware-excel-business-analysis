# 📌 Project Overview

This project presents an end-to-end Sales Analytics solution developed using **Microsoft Excel** to analyze AtliQ Hardware's sales performance and transform raw business data into meaningful insights.

Using **Power Query, Power Pivot, DAX, Pivot Tables, and Pivot Charts**, interactive dashboards and business reports were created to support data-driven decision-making across customers, markets, products, and profitability.

---

# 🎯 Business Objectives

- Analyze customer sales performance
- Evaluate market performance against sales targets
- Analyze Profit & Loss (P&L)
- Identify revenue concentration risk
- Identify high-growth customers

- Support strategic business decisions through interactive reporting
